# Fonts

This project uses the Inter font family. Download and place the following files here:

- `Inter-Regular.ttf`
- `Inter-Medium.ttf`
- `Inter-SemiBold.ttf`
- `Inter-Bold.ttf`

## Download

You can get Inter from [Google Fonts](https://fonts.google.com/specimen/Inter).

Alternatively, install via Expo:

```bash
npx expo install @expo-google-fonts/inter
```

Then update the font loading in `src/app/App.tsx` to use the expo-google-fonts package.
